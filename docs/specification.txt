================================================================================
setup-os — Dictionnaire de la configuration (glossaire dense)
================================================================================

A
---
allowUnfree (nixpkgs.config)
  Autorise les paquets non libres (propriétaires). Requis pour NVIDIA, Chrome, etc.

antigravity (home.file.".local/bin/antigravity")
  Wrapper script qui exécute /etc/nixos/config/bin/antigravity. Permet d’invoquer le binaire système depuis le profil utilisateur.

appimage-run (environment.systemPackages)
  Exécution de binaires AppImage sans installation système.

assets/
  Dossier racine : médias pour le README (logo, screenshots, wallpaper, hero-video).

auto-optimise-store (nix.settings)
  Active l’optimisation automatique du store Nix (liens en dur pour économiser l’espace).


B
---
boot.blacklistedKernelModules
  Liste de modules noyau désactivés. Ici : iTCO_wdt, iTCO_vendor_support (éviter conflits watchdog).

boot.kernelModules
  Modules chargés au boot : i2c-dev, i2c-i801 (bus I²C pour matériel type capteurs).

boot.kernelParams
  Paramètres noyau : nvidia-drm.modeset=1, pcie_aspm=off.

boot.loader.systemd-boot
  Chargeur de démarrage. configurationLimit = 1, editor = false, timeout = null. Entrée Windows 11 via extraEntries."windows.conf".

build (fileSystems."/build")
  Point de montage bind vers /home/nix-build pour le répertoire de build Nix (sandbox).


C
---
config/
  Arborescence des fichiers de configuration sources (hypr, waybar, rofi, foot, doom, bin, swappy).

config/bin/
  Scripts personnalisés : antigravity, cursor, dw-apply, dw-daemon, dw-toggle, dw-toggle-global, edex-conky-toggle, edex-toggle, hypr-plugins-init, legion-pulse, legion-toggle.

configuration.nix
  Point d’entrée système NixOS. Importe hardware-configuration.nix et les modules listés dans imports.

cursor (home.file.".local/bin/cursor")
  Symlink hors store vers /etc/nixos/config/bin/cursor (éditeur Cursor).


D
---
databases.nix (modules/)
  Module optionnel : outils et services de bases de données locales.

devShells (flake outputs)
  Coquilles de développement. .#ai : Python 3.11, pip, NVIDIA libs. .#embedded : Rust (rust-src, rust-analyzer), gcc, clang, cmake, gdb, arduino-ide/cli, esptool, openocd, minicom, mosquitto.

direnv (programs.direnv)
  Chargement automatique d’environnements par répertoire. nix-direnv activé.

download-buffer-size (nix.settings)
  Taille du buffer de téléchargement Nix (268435256 octets).

dw-apply, dw-toggle, dw-toggle-global, dw-daemon (config/bin + home.file)
  Scripts et service utilisateur pour Hyprland DarkWindow (teinte/fenêtres sombres). dw-daemon : systemd user service, wantedBy graphical-session.target.


E
---
edex.nix (modules/)
  Module optionnel (non importé dans configuration.nix actuel) : configuration edex.

ELECTRON_OZONE_PLATFORM_HINT (home.sessionVariables)
  "x11" — force Electron à utiliser X11 pour compatibilité Wayland.

emacs.nix (modules/)
  Configuration Emacs / Doom (référencé dans config/doom).

environment.sessionVariables
  NIXOS_OZONE_WL = "1", EDITOR = "vim".

environment.systemPackages
  Liste des paquets installés au niveau système (python313+pydantic+anyio, adwaita-icon-theme, bibata-cursors, brightnessctl, appimage-run, fuse, bash, vim, neovim, git, wget, curl, jq, lsof, iw, ethtool, pciutils, usbutils, tree, ripgrep, fd, fzf, fastfetch, btop, htop, kitty, foot, firefox, google-chrome, wl-clipboard, pavucontrol, networkmanager, polkit_gnome, nix-ld, mesa, libglvnd, libdrm).

experimental-features (nix.settings)
  "nix-command" et "flakes".


F
---
fileSystems."/build"
  Montage bind /home/nix-build -> /build pour le sandbox Nix.

flake.lock
  Lockfile des entrées du flake (révisions exactes). À conserver pour reproductibilité.

flake.nix
  Déclaration des inputs (nixpkgs, nixpkgs-stable, home-manager, rust-overlay, hyprchroma, nix-snapd) et des outputs (nixosConfigurations.nixos, homeConfigurations.tco, devShells).

foot (config/foot/, home.file.".config/foot")
  Configuration du terminal Foot (foot.ini). Thème et palette couleurs.

Flake inputs
  nixpkgs (nixos-unstable), nixpkgs-stable (24.11), home-manager, rust-overlay, hyprchroma, nix-snapd. home-manager et rust-overlay suivent nixpkgs.

flatpak (services.flatpak)
  Support Flatpak activé.


G
---
GDM (services.displayManager.gdm)
  Gestionnaire de connexion. Permet le choix de session Hyprland ou GNOME.

gnome (services.desktopManager.gnome)
  Bureau GNOME activé.

guix (services.guix)
  Service Guix activé. Overlay flake : guix fourni par pkgs-stable.guix.

GTK (home: gtk)
  theme name = "Adwaita-dark", iconTheme = "Papirus-Dark", cursorTheme = "Bibata-Modern-Ice".


H
---
hardware-configuration.nix
  Configuration matérielle générée (nixos-generate-config). Importé par configuration.nix.

home-manager (flake input + nixos module)
  Gestion de l’environnement utilisateur. useGlobalPkgs = true, useUserPackages = true. users.tco = ./home/tco/home.nix.

home.sessionPath
  .lmstudio/bin, .npm-global/bin, .local/bin.

home.sessionVariables
  QT_QPA_PLATFORM = "wayland", QT_QPA_PLATFORMTHEME = "qt6ct", QT_STYLE_OVERRIDE = "kvantum", XDG_DATA_DIRS, ELECTRON_OZONE_PLATFORM_HINT = "x11".

home/tco/home.nix
  Point d’entrée Home Manager : utilisateur tco, homeDirectory = /home/tco, stateVersion = "25.05", sessionPath, sessionVariables, xdg, home.file, home.packages, gtk, programs (starship, vscode, git, bash), activations, desktop entries, systemd user services.

hostName (networking)
  "nixos".

hyprchroma (flake input)
  Overlay/thème Hyprland (github:alexhulbert/hyprchroma). Utilisé pour cohérence visuelle.

Hyprland (programs.hyprland)
  enable = true, xwayland.enable = true. Config dans config/hypr (hyprland.conf, theme/seaglass.conf, tokens.conf, waybar).

imports (configuration.nix)

rofi (config/rofi, home.file.".config/rofi")

rust-overlay (flake input)
  Overlay Rust (oxalica/rust-overlay). Utilisé dans pkgs et dans devShell embedded (rust-bin.stable.latest).

snap (services.snap)

Commandes
================================================================================
  Apply system config   sudo nixos-rebuild switch --flake .#nixos
  Home Manager         home-manager switch --flake .#tco   (ou alias: hm)
  Rebuild (alias)      rebuild (bash)
  AI dev shell         nix develop .#ai   (ou devai)
  Embedded dev shell   nix develop .#embedded   (ou devemb)
  Refresh metrics      nix shell nixpkgs#cloc -c cloc . --exclude-dir=.git,node_modules,result,.direnv --md --out=docs/cloc-report.md
